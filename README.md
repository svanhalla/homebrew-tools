# homebrew-tools

Homebrew tap for svanhalla tools.

## Installation

Add the tap:
```bash
brew tap svanhalla/tools
```

Install tools:
```bash
brew install --cask <tool-name>
```

## Available Tools

| Tool | Version | Description |
|------|---------|-------------|
| **cheatd** | v1.3.7 | Personal cheat sheet management CLI with web interface |
| **svanhalla-cli** | v1.2.15 | Svanhalla CLI reklamprogram |

### cheatd
```bash
brew install --cask svanhalla/tools/cheatd
```

### svanhalla-cli
```bash
brew install --cask svanhalla/tools/svanhalla-cli
```
